# Code of Conduct

This project should be a professional, focused place to discuss code quality
tooling.

## Expected Behaviour

- Be respectful and direct.
- Assume good intent, but keep technical claims evidence-based.
- Keep issues and pull requests focused on the work.
- Be patient with users reporting false positives or confusing diagnostics.

## Unacceptable Behaviour

- Harassment, threats, or personal attacks.
- Publishing private information without permission.
- Dismissing security reports or vulnerability evidence in public threads.
- Repeated off-topic comments after maintainers ask for focus.

## Enforcement

Maintainers may edit or remove comments, close issues, block accounts, or take
other moderation actions needed to keep the project usable.

Security issues should follow [SECURITY.md](SECURITY.md), not public threads.
