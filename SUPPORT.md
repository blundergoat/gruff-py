# Support

gruff-py is pre-1.0 software.

## Getting Help

Start with:

```bash
gruff-py --help
gruff-py analyse --help
gruff-py dashboard --help
```

Useful docs:

- [Configuration](docs/CONFIGURATION.md)
- [Reporting](docs/REPORTING.md)
- [Dashboard](docs/DASHBOARD.md)
- [Rules](docs/RULES.md)

## Filing Issues

For a useful bug report, include:

- gruff-py version
- Python version
- operating system
- command that failed
- minimal project or fixture that reproduces the issue
- output with secrets removed

For false positives, include the rule id and a short code sample.

## Security Reports

Use [SECURITY.md](SECURITY.md) for vulnerability reports.
